# Active Directory Home Lab

## Introduction
This project is a virtualized Active Directory home lab.

## Technologies Used
- Windows Server 2022
- Oracle VirtualBox
- Active Directory Domain Services (AD DS)
- DNS
- SMB File Sharing
- NTFS Permissions

## Skills Demonstrated
- Configured and deployed a Domain Controller using Windows Server 2022
- Created and managed domain-joined Windows client systems
- Configured IPv4 addressing and subnetting
- Created and managed Active Directory users, groups, and organizational units
- Configured SMB shared folders and NTFS permissions
- Implemented role-based access control
- Configured isolated virtual networking using Host-Only networking in VirtualBox

## Resolved Issues
- Resolved DNS configuration issues preventing successful domain joins
- Configured an isolated virtual network to prevent interference with the home network
